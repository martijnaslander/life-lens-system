# The Life Lens System

A six-layer synaptic architecture for personal knowledge.

Website: https://martijnaslander.github.io/life-lens-system/

## Structure

- `docs/` - GitHub Pages site
- `docs/blog/` - Blog posts
- `docs/blog/feed.xml` - RSS feed

## Local development

Open `docs/index.html` in a browser.

## Deploy

Push to `main` branch. GitHub Pages serves from `docs/` folder.
